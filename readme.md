Situs transparansi pengelolaan dana online

</h2>

**SIMPADE - Simpanan Dana Desa** - merupakan website transparansi anggaran yang dibuat untuk membantu masyarakat desa mengakses informasi terkait pengelolaan dan penggunaan dana desa secara terbuka, jelas, dan mudah dipahami, sehingga dapat meningkatkan akuntabilitas pemerintah desa serta mendorong partisipasi masyarakat dalam pengawasan pembangunan desa.
